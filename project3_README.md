## What It Does

Upload any CSV file → Get an AI-generated professional executive report instantly.

The app automatically:
- Analyses your dataset (stats, patterns, correlations)
- Generates 4 custom charts
- Uses Claude AI to write a full executive report
- Provides specific business recommendations
- Lets you download the report

## How to Run Locally

```bash
# 1. Clone repo
git clone https://github.com/shreyasingh20001126/ai-insight-generator

# 2. Install dependencies
pip install -r requirements.txt

# 3. Get free API key
# Go to: console.anthropic.com → Sign up → API Keys → Create key

# 4. Run app
streamlit run project3_app.py
```

---

## How to Use

1. Open the app
2. Paste your Anthropic API key in the sidebar
3. Upload any CSV file
4. Optionally describe your business context
5. Click "Generate AI Executive Report"
6. Download the report
